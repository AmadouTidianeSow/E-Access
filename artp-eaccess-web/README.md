# Projet Angular - Version 15.2.0

Ce projet est développé avec Angular **v15.2.0**. Assurez-vous de respecter les prérequis ci-dessous pour garantir un fonctionnement correct de l'application.

## Prérequis

- **Node.js** : version **18.20.2**
- **npm** : installé avec Node.js

> Veillez à utiliser **la version exacte** de Node.js pour éviter les problèmes de compatibilité.

## Installation

1. Clonez le projet :
   ```bash
   git clone https://gitlab.com/artp-eaccess/artp-eaccess-web.git
   cd artp-eaccess-web

2. Installez les dépendances du projet :
    npm install

3. Installez la version spécifique de Webpack :
    npm install webpack@5.76.1

## Vous pouvez démarrer l'application avec l'une des deux commandes suivantes :

npm start

ou

ng serve

## Une fois le serveur démarré, l'application sera accessible à l'adresse :

http://localhost:4200
